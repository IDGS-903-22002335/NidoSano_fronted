Bienvenido a NidoSano

Tu sistema inteligente para el cuidado y monitoreo de gallineros domésticos. Este manual te ayudará a utilizar todas las funciones de la aplicación y el sistema para mantener a tus gallinas en las mejores condiciones.

1. Pantalla Principal de Monitoreo

  • Temperatura: muestra el valor actual dentro del gallinero.
  • Humedad: indica el porcentaje de humedad interna.
  • Iluminación: nivel de luz detectado.
  • Estado general: un ícono resume la situación actual del gallinero.

2. Control de Ventilación e Iluminación

  • Modo Automático: el sistema regula la ventilación e iluminación según los valores óptimos preconfigurados.
  • Modo Manual: tú decides cuándo encender o apagar ventilación e iluminación.

Tip: usa el modo manual para intervenciones específicas y vuelve a automático para evitar sobrecalentamiento o enfriamiento.

3. Alimentación y Agua

  • Visualiza el nivel actual de alimento y agua.
  • Activa una dosis manual de alimento desde la app con un solo toque.
  • Recibirás alertas cuando el nivel sea bajo.

4. Alertas y Notificaciones

Tipos de alertas:

  • Temperatura fuera de rango
  • Humedad extrema
  • Falta de alimento o agua
  • Fallo de sensor

Tip: Cada alerta incluirá recomendaciones para actuar de inmediato.

5. Historial y Reportes

Consulta el historial de condiciones ambientales y consumo. Esto te permitirá:

  • Detectar patrones estacionales.
  • Evaluar el rendimiento del sistema.
  • Planificar ajustes en ventilación, iluminación y alimentación.

Preguntas Frecuentes

¿Puedo usar el sistema sin conexión a Internet?

Sí, pero no recibirás notificaciones en tiempo real.

¿Cada cuánto debo limpiar los sensores?

Una vez al mes o si detectas lecturas anómalas.
